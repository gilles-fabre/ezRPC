Detailed Description

A combo box that lets the user choose any installed look-and-feel.

The LookAndFeelComboBox() constructor takes care of finding installed look-and-feels and putting their friendly names into the combo box.

The changeLookAndFeel() method changes the look-and-feel to the one that is currently selected in the combo box.

String reporter.LookAndFeelComboBox.changeLookAndFeel ( )

If the look-and-feel currently selected in this combo box differs from the currently active look-and-feel, then activate the selected look-and-feel.

Note that if you change the look-and-feel, then you should also apply SwingUtilities#updateComponentTreeUI(java.awt.Component) to all top-level containers so that they reflect the change.

Returns
the class name of the new look and feel, or null if the look and feel has not changed.
