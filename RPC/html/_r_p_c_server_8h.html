class  RPCServer
 An instance of RPCServer provides RPCClients with an address[:port] to connect to, over a given transport, and obtain a dedicated link (one per RPCClient). The latter can be used by the RPCClient to remotely invoke procedures of the (linked) RPCServer. Procedures can be dynamically added or removed from an RPCServer instance. See RemoteProcedureCall to get the details of the remote procedure invocation. More...
