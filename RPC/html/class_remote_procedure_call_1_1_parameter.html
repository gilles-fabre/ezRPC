Detailed Description

template<typename T>
class RemoteProcedureCall::Parameter< T >

Instances of this class are passed to the caller of the DeserializeCall method, describing and providing access to the parameters passed through the call stream built by SerializeCall. Out parameters can be modified and their modified value will transparently be passed back to the SerializeCall caller.
