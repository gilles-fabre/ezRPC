Detailed Description

Defines a dynamically programmable thread safe state machine. Only the DoTransition method is thread safe. The state machine can't be concurrently built/destroyed.

void StateMachine::InitiateTransitionTimer ( unsigned long  milli_sec_timeout,
const string &  transition_name 
)

Creates and starts a timer which will inject a transition into the current state when the given time has elapsed.

Clears the current transition timer (timeout) if any, then sets a new transition timer to the the given transition.

Parameters
milli_sec_timeoutis the delay before the transition is injected in the state machine
transition_nameis the injected transition's identifier
milli_sec_timeoutis the new timer duration
transition_nameis the name of the target transition (upon timer tiggering)
