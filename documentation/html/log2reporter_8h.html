◆ SetLogAddress()

void SetLogAddress ( const char * ipAddress,
uint16_t port )

set the IP (either IPV4 XXX.XXX.XXX.XXX numeric form or host name) address of the log-reporter server. This function overrides the settings file's address

Log Settings File:

you can create a <appname>.log-settings file in the program directory with the following keys/values:

MAX_LOG_SIZE=S SERVER_IP=H SERVER_PORT=P

where: S is the maximum size of the log file for this program, in KBytes H is the IPv4 numeric address of the LogReporter P is the Port on which the LogReporter server is listening

Parameters
ipAddressis the IP address of the log-reporter server
portis the associated port number
